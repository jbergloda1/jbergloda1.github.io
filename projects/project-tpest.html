T-Pest System

Position: Lead Mobile Developer

Team size: 6

Description: Mobile and web-based application for the Plant Protection Department of Vietnam, supporting pest monitoring and reporting. The project is architected as a multi-platform solution with a shared codebase for both React Native (mobile) and React (web), ensuring maintainability and consistency across platforms.

Responsibilities

  • Led the development of mobile and web apps using React Native and React with a modular, well-organized source code structure.
  • Implemented React Query for efficient data fetching, caching, and synchronization between client and server.
  • Designed and implemented TypeScript-based data models for type safety and scalability.
  • Integrated Ant Design components for a modern, consistent, and user-friendly web UI.
  • Coordinated with government stakeholders for requirements gathering and iterative delivery.
  • Ensured app reliability, performance, and seamless user experience across devices.

Technologies

  • React Native (Mobile)
  • React (Web)
  • TypeScript
  • React Query
  • Ant Design
  • Modular Code Architecture
