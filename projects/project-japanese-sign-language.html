Japanese Sign Language Recognition

Position: Mobile Developer

Team size: 3

Description: Speech-to-text system for Japanese Sign Language (Toppa Solution), enabling real-time translation for accessibility. Includes a dedicated Flutter app version for Android TV, supporting hands-free operation and large-screen accessibility for hearing-impaired users.

Responsibilities

  • Developed mobile and TV apps using Flutter, including a custom UI for Android TV.
  • Integrated Speech-to-Text API and implemented sign language recognition algorithms.
  • Designed and implemented background services for continuous audio processing and recognition.
  • Implemented overlay service for real-time translation display over other apps.
  • Wrote native code for high-performance audio recording and processing.
  • Integrated WebSocket for real-time communication with backend services.
  • Applied advanced techniques to prevent ANR (Application Not Responding) and reduce app crashes.
  • Worked with AngularJS for admin dashboard features.
  • Tested and improved recognition accuracy with real users on both mobile and TV platforms.

Technologies

  • Flutter (Mobile & TV)
  • Android TV
  • Background Service
  • Overlay Service
  • Native Android (Kotlin/Java) for audio record
  • Speech-to-Text API
  • WebSocket
  • Crash/ANR prevention
  • AngularJS
